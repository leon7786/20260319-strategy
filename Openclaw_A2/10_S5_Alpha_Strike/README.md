# S5 Alpha Strike

## 设计目标
- 追求更高收益，同时尽量控制回撤不恶化
- 作为 S1 的进攻升级版本

## 回测约束（严格）
- 使用真实日线：`IXIC + VUSTX`
- **t-1 生成信号，t 执行**（无未来函数）
- 纳入摩擦：
  - 滑点 `0.08%`
  - 权益持仓成本 `0.15%/月`
  - 债券持仓成本 `0.06%/月`
  - 杠杆借贷成本 `5%/年`

## 参数
见 `strategy.py` 中 `PARAMS`。

## 实测结果（1995-2025）
- CAGR: **31.61%**
- MaxDD: **-48.45%**
- Sharpe: **0.95**

## OOS（2015-2025）

- MaxDD: **-48.45%**
- Sharpe: **0.82**
