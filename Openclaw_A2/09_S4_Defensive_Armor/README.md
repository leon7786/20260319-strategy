# S4 Defensive Armor

## 设计目标
- 在保持较高收益的前提下进一步压低回撤
- 适合更重视净值曲线稳定性的场景

## 回测约束（严格）
- 使用真实日线：`IXIC + VUSTX`
- **t-1 生成信号，t 执行**（无未来函数）
- 纳入摩擦：
  - 滑点 `0.08%`
  - 权益持仓成本 `0.15%/月`
  - 债券持仓成本 `0.06%/月`
  - 杠杆借贷成本 `5%/年`

## 参数
见 `strategy.py` 中 `PARAMS`。

## 实测结果（1995-2025）
- CAGR: **25.90%**
- MaxDD: **-43.31%**
- Sharpe: **0.94**

## OOS（2015-2025）
- CAGR: **23.66%**
- MaxDD: **-41.59%**
- Sharpe: **0.86**

## 运行
```bash
python3 strategy.py
```
