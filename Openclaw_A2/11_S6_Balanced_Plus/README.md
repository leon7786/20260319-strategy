# S6 Balanced Plus

## 设计目标
- 在收益与回撤之间做更平衡的折中
- 作为 S4 / S5 之间的中间档

## 回测约束（严格）
- 使用真实日线：`IXIC + VUSTX`
- **t-1 生成信号，t 执行**（无未来函数）
- 纳入摩擦：
  - 滑点 `0.08%`
  - 权益持仓成本 `0.15%/月`
  - 债券持仓成本 `0.06%/月`
  - 杠杆借贷成本 `5%/年`

## 参数
见 `strategy.py` 中 `PARAMS`。

## 实测结果（1995-2025）
- CAGR: **28.60%**
- MaxDD: **-47.87%**
- Sharpe: **0.93**

## OOS（2015-2025）
- CAGR: **23.47%**
- MaxDD: **-46.28%**
- Sharpe: **0.80**

## 运行
```bash
python3 strategy.py
```
