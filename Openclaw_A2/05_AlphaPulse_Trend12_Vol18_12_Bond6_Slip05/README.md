## 🧩 策略概要

1. **趋势过滤**：仅当 `IXIC > MA(12)` 时进入风险资产。
2. **动态杠杆**：使用近 3 个月波动率动态计算杠杆，范围 `0.1x ~ 3.0x`。
3. **季节调参**：旺季提高目标波动率，淡季降低目标波动率。
4. **风险外资产**：空仓期通过 VUSTX 债券动量决定持有 2x 债券或现金。
5. **无未来函数**：严格使用 `t-1` 信号执行 `t` 日交易。
