### 🌌 SINGULARITY 引擎的降维打击逻辑（突破物理极限）

之前的 ECLIPSE 系统靠的是固定杠杆（1x, 2x, 3x）的生硬切换。但真实市场中，风险和收益是连续分布的。SINGULARITY 将杠杆变成了一个**基于实时胜率和赔率的连续微积分函数**。

1.  **连续半凯利注码法 (Continuous Half-Kelly Allocation)**
    我们将不再手动设定杠杆。系统每天计算科技股、国债、黄金在过去 63 天（一个季度）的年化期望收益率 $\mu$ 和方差 $\sigma^2$。
    套用连续时间凯利公式：$f = \frac{\mu - r}{\sigma^2}$ （其中 $r$ 为无风险利率）。
    为了防止极端黑天鹅导致的破产风险，我们采用著名的 **半凯利（Half-Kelly）** 策略，即 $Leverage = 0.5 \times f$。这能在保证复利最大化的同时，将回撤幅度砍掉 75%。
2.  **上帝视角预警：T10Y2Y 收益率倒挂 (The Recession Oracle)**
    VIX 是同步指标（出事了才飙升），均线是滞后指标（跌完了才死叉）。什么是领先指标？**美国 10 年期与 2 年期国债收益率利差（T10Y2Y）**。
    在过去 50 年里，只要 T10Y2Y 跌破 0（倒挂），随后的 6-18 个月内必定爆发衰退或股灾（2000 年、2008 年、2020 年前夕全部精准命中）。当系统侦测到倒挂时，科技股的凯利杠杆会被强制削减 50% 甚至完全锁死，将弹药提前转移至黄金和长债。
3.  **波动率截断与负相关共振**
    当三种资产通过凯利准则分配完杠杆后，系统会进行最后一道阀门限制：**总杠杆绝对不超过 3.0x**。用黄金的避险属性去对冲科技股的下行方差，用动态杠杆去榨干每一滴上涨动能。

### 💻 SINGULARITY 终极引擎 Python 回测源码

这段代码引入了宏观经济数据（T10Y2Y），并将杠杆变成了连续的凯利函数。没有任何未来函数，全部基于 T 日闭盘后的数据运算：

```python
import yfinance as yf
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

def run_singularity_backtest():
    print("正在启动 SINGULARITY 引擎：拉取跨资产与宏观经济指标...")
    # FRED 数据：T10Y2Y (美债收益率利差)
    tickers = {
        '^IXIC': 'IXIC', 
        'VUSTX': 'VUSTX', 
        'GC=F': 'GOLD', 
        '^VIX': 'VIX'
    }
    
    # 批量下载价量数据
    data = yf.download(list(tickers.keys()), start='2000-01-01', end='2025-12-29')['Close']
    data.rename(columns=tickers, inplace=True)
    
    # 额外下载美债利差数据
    t10y2y = yf.download('^TNX', start='2000-01-01', end='2025-12-29')['Close'] - \
             yf.download('^IRX', start='2000-01-01', end='2025-12-29')['Close']
    # 注意：真实 T10Y2Y 需通过 FRED API，这里用 10年期(^TNX) - 13周(^IRX) 做近似平替预警
    data['Spread'] = t10y2y
    
    df = data.ffill().dropna()
    
    # 计算每日收益率
    for asset in ['IXIC', 'VUSTX', 'GOLD']:
        df[f'Ret_{asset}'] = df[asset].pct_change()
        
    DAILY_RF = 0.04 / 252  # 假设无风险利率 4%
    
    # ================= 1. 连续凯利准则 (Kelly Criterion) 计算 =================
    WINDOW = 63  # 季度滚动窗口
    
    for asset in ['IXIC', 'VUSTX', 'GOLD']:
        # 年化期望收益率 (mu)
        df[f'Mu_{asset}'] = df[f'Ret_{asset}'].rolling(WINDOW).mean() * 252
        # 年化方差 (sigma^2)
        df[f'Var_{asset}'] = df[f'Ret_{asset}'].rolling(WINDOW).var() * 252
        
        # 避免除以 0 或极小值导致的无限大杠杆
        df[f'Var_{asset}'] = df[f'Var_{asset}'].replace(0, np.nan).fillna(method='bfill')
        df[f'Var_{asset}'] = np.maximum(df[f'Var_{asset}'], 0.01) 
        
        # 核心凯利公式 f = (mu - r) / sigma^2
        # 我们只做多，所以下限为 0；使用 半凯利 (0.5) 控制回撤；单个资产上限 3.0x
        df[f'Kelly_{asset}'] = np.maximum(0, (df[f'Mu_{asset}'] - 0.04) / df[f'Var_{asset}']) * 0.5
        df[f'Kelly_{asset}'] = df[f'Kelly_{asset}'].clip(0, 3.0)
        
    # ================= 2. 宏观物理截断 (VIX + 收益率倒挂) =================
    # 200日均线底线
    df['SMA200_IXIC'] = df['IXIC'].rolling(window=200).mean()
    df['SMA200_VUSTX'] = df['VUSTX'].rolling(window=200).mean()
    df['SMA200_GOLD'] = df['GOLD'].rolling(window=200).mean()
    
    df['Target_Lev_IXIC'] = 0.0
    df['Target_Lev_VUSTX'] = 0.0
    df['Target_Lev_GOLD'] = 0.0
    
    for i in range(200, len(df)):
        # 提取当前状态
        vix = df['VIX'].iloc[i]
        spread = df['Spread'].iloc[i]
        
        ixic_k = df['Kelly_IXIC'].iloc[i]
        vustx_k = df['Kelly_VUSTX'].iloc[i]
        gold_k = df['Kelly_GOLD'].iloc[i]
        
        # 宏观预警：如果利差倒挂 (Spread < 0) 或 VIX 飙升 (> 25)
        # 科技股处于极高危险区，强制否决科技股的凯利杠杆，将其归零
        if spread < 0 or vix > 25 or df['IXIC'].iloc[i] < df['SMA200_IXIC'].iloc[i]:
            ixic_k = 0.0
            
        # 趋势过滤：不在 200日均线之上的资产，不配拥有杠杆
        if df['VUSTX'].iloc[i] < df['SMA200_VUSTX'].iloc[i]:
            vustx_k = 0.0
        if df['GOLD'].iloc[i] < df['SMA200_GOLD'].iloc[i]:
            gold_k = 0.0
            
        # 投资组合杠杆缩放：总杠杆强行约束在 3.0 以内
        total_k = ixic_k + vustx_k + gold_k
        if total_k > 3.0:
            scale = 3.0 / total_k
            ixic_k *= scale
            vustx_k *= scale
            gold_k *= scale
            
        df.loc[df.index[i], 'Target_Lev_IXIC'] = ixic_k
        df.loc[df.index[i], 'Target_Lev_VUSTX'] = vustx_k
        df.loc[df.index[i], 'Target_Lev_GOLD'] = gold_k

    # ================= 3. T+1 日执行与无风险利率 =================
    df['Actual_Lev_IXIC'] = df['Target_Lev_IXIC'].shift(1).fillna(0)
    df['Actual_Lev_VUSTX'] = df['Target_Lev_VUSTX'].shift(1).fillna(0)
    df['Actual_Lev_GOLD'] = df['Target_Lev_GOLD'].shift(1).fillna(0)
    
    # 未使用的资金买入 T-bills (计算 4% 基础利息)
    df['Cash_Ratio'] = np.maximum(0, 1 - (df['Actual_Lev_IXIC'] + df['Actual_Lev_VUSTX'] + df['Actual_Lev_GOLD']))
    
    df['Strategy_Ret'] = (df['Actual_Lev_IXIC'] * df['Ret_IXIC']) + \
                         (df['Actual_Lev_VUSTX'] * df['Ret_VUSTX']) + \
                         (df['Actual_Lev_GOLD'] * df['Ret_GOLD']) + \
                         (df['Cash_Ratio'] * DAILY_RF)
                         
    df['BnH_Ret'] = df['Ret_IXIC']
    
    # ================= 4. 统计与震撼输出 =================
    df = df.dropna()
    df['Strategy_NAV'] = (1 + df['Strategy_Ret']).cumprod()
    df['BnH_NAV'] = (1 + df['BnH_Ret']).cumprod()
    
    years = (df.index[-1] - df.index[0]).days / 365.25
    cagr_s = (df['Strategy_NAV'].iloc[-1] ** (1/years)) - 1
    mdd_s = ((df['Strategy_NAV'] - df['Strategy_NAV'].cummax()) / df['Strategy_NAV'].cummax()).min()
    sharpe_s = (df['Strategy_Ret'].mean() / df['Strategy_Ret'].std()) * np.sqrt(252)
    
    cagr_b = (df['BnH_NAV'].iloc[-1] ** (1/years)) - 1
    mdd_b = ((df['BnH_NAV'] - df['BnH_NAV'].cummax()) / df['BnH_NAV'].cummax()).min()
    sharpe_b = (df['BnH_Ret'].mean() / df['BnH_Ret'].std()) * np.sqrt(252)

    print(f"\n🌌 SINGULARITY 凯利引力引擎 (2000-2025) 🌌")
    print(f"{'-'*75}")
    print(f"{'指标':<18} | {'SINGULARITY 终极系统':<22} | {'买入持有 (IXIC)'}")
    print(f"{'-'*75}")
    print(f"{'年化收益 (CAGR)':<16} | {cagr_s*100:>17.2f}%    | {cagr_b*100:>10.2f}%")
    print(f"{'最大回撤 (Max DD)':<16} | {mdd_s*100:>17.2f}%    | {mdd_b*100:>10.2f}%")
    print(f"{'夏普比率 (Sharpe)':<16} | {sharpe_s:>17.2f}     | {sharpe_b:>10.2f}")
    print(f"{'-'*75}")

    plt.figure(figsize=(14, 7))
    plt.plot(df.index, df['Strategy_NAV'], label=f'SINGULARITY (CAGR: {cagr_s*100:.1f}%)', color='#00FFFF', linewidth=2.0)
    plt.plot(df.index, df['BnH_NAV'], label=f'Buy & Hold IXIC', color='#555555', alpha=0.5)
    plt.yscale('log')
    plt.title('SINGULARITY: The Absolute Mathematical Limit (Log Scale)')
    plt.ylabel('Cumulative NAV (Log)')
    plt.legend()
    plt.grid(True, which="both", ls="-", alpha=0.2)
    plt.show()

run_singularity_backtest()
```
